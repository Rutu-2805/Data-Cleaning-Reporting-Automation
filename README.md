# Data-Cleaning-Reporting-Automation

## Project Overview
It is a Python-based project that automates data preprocessing, removes inconsistencies, and generates insightful reports and visualizations for efficient data analysis.

## Features
- Handle missing values
- Remove duplicate records
- Standardize inconsistent text values
- Generate automated reports
- Create visualizations automatically

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Run the Project
python main.py
